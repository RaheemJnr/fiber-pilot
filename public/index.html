⚡ fiber-pilot Setup Guide

Prerequisites

  • Node.js 20+
  • Fiber Network node binary fnn v0.7.1+ — releases
  • CKB testnet coins — faucet.nervos.org
  • Anthropic API key with credits (for the web UI)

1. Start your Fiber Node

Always run from inside your fiber-node directory with -d . so Fiber finds your local key file:

cd ~/path/to/fiber-node
FIBER_SECRET_KEY_PASSWORD='your-password' ./fnn -d . -c config.yml

⚠ Key format: Fiber needs a raw 32-byte hex key (64 chars, one line). If you exported with ckb-cli --extended-privkey-path, it writes two lines — keep only the first:

head -1 ckb/key > ckb/key.tmp && mv ckb/key.tmp ckb/key
chmod 600 ckb/key

2. Use the Web Chat (this page)

The simplest way — enter your node's RPC URL below and start chatting.

To run your own instance:

git clone https://github.com/RaheemJnr/fiber-pilot
cd fiber-pilot && npm install && npm run build

# Create .env
echo "ANTHROPIC_API_KEY=sk-ant-api03-YOUR_KEY" > .env
echo "PORT=3000" >> .env

npm run server   # opens http://localhost:3000

⚠ Get a proper API key at console.anthropic.com → Settings → API Keys (starts with sk-ant-api03-). The token Claude Code uses (sk-ant-oat01-) won't work.

3. Use as MCP Server — Hosted (no install)

The hosted server exposes a /mcp HTTP endpoint. Claude Code connects directly — no cloning, no billing (uses your Claude subscription).

Add to your project's .mcp.json:

{
  "mcpServers": {
    "fiber-pilot": {
      "type": "http",
      "url": "https://HOSTED-SERVER/mcp?rpc=http://YOUR-FIBER-NODE-IP:8227"
    }
  }
}

⚠ Your Fiber node's RPC port (8227) must be reachable from the hosted server. Use a VPS for your Fiber node, or port-forward your router. Do not expose without a firewall on mainnet.

4. Use as MCP Server — Local (fully private)

Clone the repo and run the MCP server locally. Your node never leaves your machine and all tool calls use your Claude Code subscription.

git clone https://github.com/RaheemJnr/fiber-pilot
cd fiber-pilot && npm install && npm run build

Add to your project's .mcp.json:

{
  "mcpServers": {
    "fiber-pilot": {
      "command": "node",
      "args": ["/absolute/path/to/fiber-pilot/dist/index.js"],
      "env": {
        "FIBER_RPC_URL": "http://127.0.0.1:8227",
        "FP_MAX_CHANNEL_OPEN": "10000",
        "FP_APPROVAL_THRESHOLD": "1000"
      }
    }
  }
}

Open Claude Code — fiber-pilot tools are available immediately. Try: "Analyze my channels" or "Open a 200 CKB channel with the best connected peer."

400 credit balance too lowAdd credits at console.anthropic.com → Billing
503 No available accountsProxy quota exhausted — check your API key or switch proxies
Decryption failed: aead::ErrorKey file has two lines (extended format) — keep only the first line
No such file or directory for keyMissing -d . flag — Fiber defaults to ~/.fiber-node
Node disconnected in UIFiber RPC not reachable — verify fnn is running and the URL is correct
RocksDB LOCK errorStale lock from crash — delete fiber/store/LOCK and restart
Resource temporarily unavailablefnn already running — pkill -9 fnn then restart
