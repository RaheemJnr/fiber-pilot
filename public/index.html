⚡ fiber-pilot Setup Guide

Prerequisites

  • Node.js 20+ (for --env-file flag support)
  • A running CKB node pointed at testnet
  • Fiber Network node binary fnn v0.7.1+
  • An Anthropic API key with credits

1. Clone & Install

git clone https://github.com/RaheemJnr/fiber-pilot
cd fiber-pilot
npm install
npm run build

2. Anthropic API Key

You need a proper API key, not the Claude Code OAuth token.

  1. Go to console.anthropic.com → Settings → API Keys
  2. Click Create Key — the key starts with sk-ant-api03-
  3. Add billing credits at Settings → Billing (even $5 works for demos)

⚠ The token Claude Code uses (sk-ant-oat01-...) is an OAuth token — it won't work here.

3. Configure .env

ANTHROPIC_API_KEY=sk-ant-api03-YOUR_KEY_HERE
FIBER_RPC_URL=http://127.0.0.1:8227
PORT=3000

This file is in .gitignore — your key stays local.

4. Start the Fiber Node

Always run from inside your fiber-node directory with -d . so Fiber uses the local key file (not ~/.fiber-node):

cd ~/path/to/fiber-node
FIBER_SECRET_KEY_PASSWORD='your-password' ./fnn -d . -c config.yml

⚠ Key format: Fiber needs a raw 32-byte hex key (64 chars). If you exported with ckb-cli --extended-privkey-path, it writes two lines — use only the first line.

# Fix extended key (keep only first line):
head -1 ckb/key > ckb/key.tmp && mv ckb/key.tmp ckb/key
chmod 600 ckb/key

5. Run fiber-pilot Web UI

npm run build && npm run server

Open http://localhost:3000, enter your Fiber node RPC URL, and connect.

6. Use as MCP Server (Claude Code / Claude Desktop)

fiber-pilot is also a fully spec-compliant MCP server — no web server or hosting needed. Claude Code spawns it as a local process and calls all 17 tools directly.

Add this to your project's .mcp.json:

{
  "mcpServers": {
    "fiber-pilot": {
      "command": "node",
      "args": ["/absolute/path/to/fiber-pilot/dist/index.js"],
      "env": {
        "FIBER_RPC_URL": "http://127.0.0.1:8227"
      }
    }
  }
}

Then in Claude Code you can say "Open a 500 CKB channel with bootnodehk" or "Analyze my channels" and Claude will call the tools autonomously — using your Claude Code subscription, no API billing needed.

⚠ The MCP server communicates over stdio (stdin/stdout). It has no HTTP port — Claude Code manages the process lifecycle automatically.

Troubleshooting

401 invalid x-api-keyKey is the Claude Code OAuth token — create a new API key at console.anthropic.com
400 credit balance too lowAdd credits at console.anthropic.com → Billing
Decryption failed: aead::ErrorKey file has two lines (extended format) — keep only the first line
No such file or directory for keyMissing -d . flag — Fiber defaults to ~/.fiber-node
Node disconnected in UIFiber RPC not reachable — check FIBER_RPC_URL and that fnn is running
RocksDB LOCK errorStale lock from crash — delete fiber/store/LOCK and retry
